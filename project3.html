Semantic Web Recipe Explorer Movie Rating Service




View on GitHub

Overview:

The Movie Ratings Service is a RESTful API that allows users to sign up, log in, and submit ratings for movies. It includes role-based access control where regular users can rate movies and view ratings, while administrators can manage movie data but are restricted from submitting ratings. JWT-based authentication ensures secure and stateless login sessions. The API also supports file uploads for movie-related assets (such as posters), with validation to accept only specific file formats.

Key Features:


  • User registration and login with secure password handling.
  • Role-based access (users vs. admins).
  • JWT (JSON Web Token) authentication for protected endpoints.
  • CRUD operations for movie ratings (Create, Read, Update, Delete).
  • Admin-only endpoints to manage the movie database.
  • File upload feature with extension validation (e.g., JPG, PNG).
  • RESTful API design, well-structured and documented.

Technologies Used:


  • Backend Framework: Flask
  • Authentication & Security: JWT (PyJWT), bcrypt, werkzeug.security
  • Database: SQLAlchemy ORM with SQLite or MySQL (configurable)
  • Environment Management: Python Virtual Environments, .env files
  • File Uploads: Flask's built-in file handling with extension filtering
  • Development Tools: Postman for API testing, Git for version control

Challenges Faced:


  • Implementing secure and stateless authentication using JWT.
  • Enforcing role-based access control with clear permission boundaries.
  • Safely handling file uploads to prevent abuse or invalid data.
  • Managing token expiration and refresh logic.
  • Designing intuitive and consistent REST endpoints for scalability.

Outcome:

The project resulted in a secure, scalable, and well-documented API service that can be used in full-stack movie review platforms or integrated into frontend clients. It demonstrates proficiency in authentication, REST architecture, and backend development best practices. The service is suitable for learning, personal use, or as a foundation for production-level systems with further enhancements like pagination, advanced search, and analytics.
